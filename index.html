About BetOS & Clash

BetOS is a custom Linux distribution tailored for experimenting and running the Clash programming language. Clash combines Bash-style commands with low-level Assembly-style operations, giving you full control over your OS and desktop apps.

Features

Custom Kernel

BetOS is built with a fully customized Linux kernel for performance and flexibility.

Clash Language

Run powerful scripts to control your OS and interact with your desktop environment.

Desktop Apps

Enjoy a full GUI with pre-installed apps, including a BetLang terminal for Clash.
